## 二、CTAResearch 研究框架

### 2.1 框架初始化

```python
from czsc.sensors import CTAResearch
from czsc.strategies import CzscStrategyBase

# 定义读取K线函数
def read_bars(symbol, freq, sdt, edt, fq='后复权'):
    """读取K线数据

    参数：
        symbol: 标的代码
        freq: K线周期
        sdt: 开始日期
        edt: 结束日期
        fq: 复权方式
    """
    from czsc.core import format_standard_kline, Freq
    from czsc.mock import generate_symbol_kines

    df = generate_symbol_kines(symbol, freq, sdt, edt)
    bars = format_standard_kline(df, freq=Freq.F30)
    return bars

# 创建 CTA 研究框架
cta = CTAResearch(
    strategy=MyStrategy,           # 策略类
    read_bars=read_bars,            # K线读取函数
    results_path='./cta_results',   # 结果保存路径
    signals_module_name='czsc.signals'
)

# 自动保存策略配置到 results_path/positions/
```

## 三、DummyBacktest 快速回测

```python
from czsc.traders import DummyBacktest

## 四、回测指标详解

### 4.1 收益指标

```python
"""
收益类指标：

1. 总收益（Total Return）
   定义：累计收益金额
   计算：sum(平仓收益)
   单位：元

2. 年化收益（Annualized Return）
   定义：年化收益率
   计算：(1 + 总收益率)^(365/持仓天数) - 1
   单位：%

3. 平均收益（Average Return）
   定义：平均每笔交易收益
   计算：sum(平仓收益) / 交易次数
   单位：元

4. 单笔最大收益（Max Single Profit）
   定义：单笔交易最大盈利
   计算：max(平仓收益)
   单位：元
"""

from czsc.utils import daily_performance

def calculate_return_metrics(pairs):
    """计算收益指标

    参数：
        pairs: 交易对列表

    返回：
        收益指标字典
    """
    import numpy as np

    pnls = [pair.pnl for pair in pairs]

    return {
        '总收益': sum(pnls),
        '平均收益': np.mean(pnls),
        '最大单笔收益': max(pnls),
        '最大单笔亏损': min(pnls),
        '收益标准差': np.std(pnls),
    }
```

### 4.2 风险指标

```python
"""
风险类指标：

1. 最大回撤（Max Drawdown）
   定义：从高点到低点的最大跌幅
   计算：max((峰值 - 当前值) / 峰值)
   单位：%

2. 收益波动率（Volatility）
   定义：收益的标准差
   计算：std(日收益率) * sqrt(252)
   单位：%

3. 下行风险（Downside Risk）
   定义：负收益的波动率
   计算：std(负收益) * sqrt(252)
   单位：%

4. VaR（Value at Risk）
   定义：置信水平下的最大损失
   计算：percentile(收益, 5%)
   单位：元
"""

from czsc.utils import top_drawdowns

def calculate_risk_metrics(daily_returns):
    """计算风险指标

    参数：
        daily_returns: 日收益率序列

    返回：
        风险指标字典
    """
    import numpy as np

    # 计算累计收益曲线
    cum_returns = (1 + daily_returns).cumprod()

    # 计算最大回撤
    peak = cum_returns.expanding(min_periods=1).max()
    drawdown = (cum_returns - peak) / peak
    max_dd = drawdown.min()

    # 计算波动率
    volatility = daily_returns.std() * (252 ** 0.5)

    return {
        '最大回撤': abs(max_dd),
        '波动率': volatility,
        '夏普比率': (daily_returns.mean() * 252) / volatility,
    }
```

### 4.3 效率指标

```python
"""
效率类指标：

1. 夏普比率（Sharpe Ratio）
   定义：单位风险的超额收益
   计算：(年化收益 - 无风险利率) / 年化波动率
   单位：无

2. 卡玛比率（Calmar Ratio）
   定义：单位回撤的年化收益
   计算：年化收益 / 最大回撤
   单位：无

3. 信息比率（Information Ratio）
   定义：相对基准的超额收益/跟踪误差
   计算：(组合收益 - 基准收益) / 跟踪误差
   单位：无

4. 盈亏比（Profit/Loss Ratio）
   定义：平均盈利与平均亏损之比
   计算：平均盈利 / abs(平均亏损)
   单位：无
"""

def calculate_efficiency_metrics(pairs, daily_returns, risk_free_rate=0.03):
    """计算效率指标"""
    import numpy as np

    # 夏普比率
    annual_return = daily_returns.mean() * 252
    volatility = daily_returns.std() * (252 ** 0.5)
    sharpe = (annual_return - risk_free_rate) / volatility

    # 盈亏比
    profits = [p.pnl for p in pairs if p.pnl > 0]
    losses = [p.pnl for p in pairs if p.pnl < 0]

    profit_loss_ratio = np.mean(profits) / abs(np.mean(losses)) if losses else 0

```

### 4.4 指标解读决策框架

> 💡 **专家视角**：指标不是孤立的数字，需要组合判断。

**第一步：排除不合格策略**（红线指标）

| 指标 | 红线 | 说明 |
|------|------|------|
| 最大回撤 | > 30% | 绝大多数投资者无法承受 |
| 交易次数 | < 30 | 统计意义不足 |
| 胜率 | < 30% 且盈亏比 < 2 | 收益无法覆盖亏损 |

**第二步：评估策略质量**（分级标准）

| 等级 | 夏普比率（Sharpe Ratio） | 卡玛比率（Calmar Ratio） | 月度胜率 |
|------|---------|---------|----------|
| 优秀 | > 2.0 | > 3.0 | > 70% |
| 良好 | 1.0-2.0 | 1.5-3.0 | 55-70% |
| 一般 | 0.5-1.0 | 0.5-1.5 | 45-55% |

**第三步：检查稳健性**
- 收益曲线是否平滑？（波动大 = 策略不稳定）
- 收益是否集中在少数交易？（依赖单次大赢 = 不可持续）
- 不同时间段表现是否一致？（仅某段好 = 过拟合风险）
    return {
        '夏普比率': sharpe,
        '盈亏比': profit_loss_ratio,
    }
```

## 六、常见问题

### Q1: 如何处理手续费和滑点？

A: 在策略中设置手续费参数：

```python
class StrategyWithFee(CzscStrategyBase):
    """包含手续费的计算"""

    def __init__(self, **kwargs):
        super().__init__(**kwargs)
        self.commission_rate = kwargs.get('commission_rate', 0.0003)  # 万三
        self.slippage_rate = kwargs.get('slippage_rate', 0.0001)      # 万一

    def calculate_actual_pnl(self, open_price, close_price, direction, amount):
        """计算扣除手续费后的实际收益"""
        # 手续费
        commission = (open_price + close_price) * amount * self.commission_rate

        # 滑点
        slippage = (open_price + close_price) * amount * self.slippage_rate

        # 实际收益
        if direction == Direction.Up:
            gross_pnl = (close_price - open_price) * amount
        else:
            gross_pnl = (open_price - close_price) * amount

        net_pnl = gross_pnl - commission - slippage
        return net_pnl
```
