## 🎯 学习成果自检

完成学习路径后，用以下清单验证你的掌握程度：

### 用户路径自检
- [ ] 能独立安装 CZSC 并验证版本
- [ ] 能用 `generate_symbol_kines` 生成测试数据并创建 `CZSC` 对象
- [ ] 能解释分型→笔→中枢的递进关系
- [ ] 能编写包含3个以上信号的策略并完成回测
- [ ] 能使用 `KlineChart` 和 `plot_cumulative_returns` 绘制分析图表

### 开发者路径自检
- [ ] 能编写符合命名规范的信号函数并通过测试
- [ ] 能解释 Rust/Python 混合架构的回退机制
- [ ] 能使用 `uv run pytest` 运行测试套件
- [ ] 能提交符合 CI 标准的 Pull Request
