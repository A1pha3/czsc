## 二、中枢参数

| 参数 | 计算方式 | 含义 |
|------|----------|------|
| **ZG** | min(前3笔 high) | 中枢上沿：前三笔高点的最小值 |
| **ZD** | max(前3笔 low) | 中枢下沿：前三笔低点的最大值 |
| **ZZ** | (ZG + ZD) / 2 | 中枢中轴：多空均衡价格 |
| **GG** | max(所有笔 high) | 中枢区间最高点 |
| **DD** | min(所有笔 low) | 中枢区间最低点 |

## 五、笔的力度指标

| 属性 | 公式 | 含义 | 值域 |
|------|------|------|------|
| `power_price` | \|fx_b.fx - fx_a.fx\| | 价格幅度 | > 0 |
| `power_volume` | Σ bars[1:-1].vol | 成交量力度 | > 0 |
| `change` | (fx_b.fx - fx_a.fx) / fx_a.fx | 涨跌幅 | 正/负 |
| `length` | len(bars) | 无包含K线数 | ≥ 6 |
| `slope` | polyfit(close, 1)[0] | 线性拟合斜率 | 正/负 |
| `rsq` | R² | 线性拟合优度 | [0, 1] |
| `SNR` | 信噪比 | 走势纯净度 | > 0 |
| `acceleration` | polyfit(close, 2)[0] | 二次拟合加速度 | 正/负 |
| `hypotenuse` | √(power_price² + len(raw_bars)²) | 斜边长度 | > 0 |
| `angle` | 斜边与竖直夹角 | 力度角度 | [0°, 90°] |

## 八、核心公式速查

```
包含关系判断:
  include = (k2.H≥k3.H ∧ k2.L≤k3.L) ∨ (k2.H≤k3.H ∧ k2.L≥k3.L)

方向判定:
  dir = Up if k1.H < k2.H else Down

向上合并: H' = max(k2.H, k3.H),  L' = max(k2.L, k3.L)
向下合并: H' = min(k2.H, k3.H),  L' = min(k2.L, k3.L)

顶分型: k1.H < k2.H > k3.H  ∧  k1.L < k2.L > k3.L
底分型: k1.H > k2.H < k3.H  ∧  k1.L > k2.L < k3.L

成笔条件: len(bars) ≥ min_bi_len  ∧  ¬ab_include(fx_a, fx_b)

中枢参数:
  ZG = min(bi[0].H, bi[1].H, bi[2].H)
  ZD = max(bi[0].L, bi[1].L, bi[2].L)
  ZZ = (ZG + ZD) / 2
  valid = ZG ≥ ZD

背驰条件:
  bc_price  = last.power_price  < ref.power_price
  bc_volume = last.power_volume < ref.power_volume
  bc_length = last.length       < ref.length
  背驰 = bc_price ∧ (bc_volume ∨ bc_length)
```
