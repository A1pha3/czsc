## 一、多级别分析概述

### 1.1 基本概念

```python
"""
多级别分析：同时分析多个时间周期的市场状态

核心思想：
1. 大级别定方向 - 判断主要趋势
2. 中级别找位置 - 确定入场位置
3. 小级别精准入场 - 选择最佳入场点

级别关系：
小级别构成大级别，大级别包含小级别
"""
```

### 1.2 级别传递图

```
┌─────────────────────────────────────────────────────────┐
│                    级别传递关系                         │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  月线                                                    │
│   └── 周线                                               │
│       └── 日线                                          │
│           └── 60分钟                                     │
│               └── 30分钟                                 │
│                   └── 5分钟                              │
│                       └── 1分钟                          │
│                                                         │
│  小级别变化 → 中级别变化 → 大级别变化                   │
│  （敏感度递减，稳定性递增）                             │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 1.3 级别选择原则

| 分析目标 | 推荐级别组合 | 说明 |
|----------|-------------|------|
| 日内交易 | 1分钟 + 5分钟 + 30分钟 | 精准入场 |
| 波段交易 | 5分钟 + 30分钟 + 日线 | 把握波段 |
| 趋势交易 | 30分钟 + 日线 + 周线 | 把握趋势 |
| 长线投资 | 日线 + 周线 + 月线 | 长期持有 |

## 二、多周期K线合成

### 2.1 基础合成

```python
from czsc.core import BarGenerator, format_standard_kline, Freq
from czsc.mock import generate_symbol_kines

# 准备1分钟基础数据
df = generate_symbol_kines('000001', '1分钟', '20240101', '20240131')
bars_1m = format_standard_kline(df, freq=Freq.F1)

# 创建多周期生成器
bg = BarGenerator(
    base_freq='1分钟',  # 基础周期
    freqs=['5分钟', '15分钟', '30分钟', '60分钟', '日线']
)

# 输入基础K线
for bar in bars_1m:
    bg.update(bar)

# 获取合成后的K线
bars_5m = bg.bars['5分钟']
bars_15m = bg.bars['15分钟']
bars_30m = bg.bars['30分钟']
bars_60m = bg.bars['60分钟']
bars_daily = bg.bars['日线']

print(f"1分钟 → 5分钟: {len(bars_1m)} → {len(bars_5m)}")
print(f"1分钟 → 日线: {len(bars_1m)} → {len(bars_daily)}")
```

```python
def get_level_status(czsc_obj):
```

## 四、级别共振分析

### 4.1 共振类型

```python
"""
级别共振类型：

1. 同向共振
   - 所有级别方向一致
   - 最强信号

2. 递进共振
   - 大级别定方向
   - 小级别确认
   - 次强信号

3. 背离共振
   - 大级别和小级别方向相反
   - 谨慎对待
"""
```

### 4.2 共振检测函数

```python
from czsc.py.enum import Direction

def detect_resonance(czsc_daily, czsc_30m, czsc_5m):
    """检测级别共振

    返回共振状态
    """
    # 获取各周期最新笔方向
    daily_dir = czsc_daily.bi_list[-1].direction if czsc_daily.bi_list else None
    m30_dir = czsc_30m.bi_list[-1].direction if czsc_30m.bi_list else None
    m5_dir = czsc_5m.bi_list[-1].direction if czsc_5m.bi_list else None

    # 同向共振检测
    if daily_dir == m30_dir == m5_dir == Direction.Up:
        return "多头共振"
    elif daily_dir == m30_dir == m5_dir == Direction.Down:
        return "空头共振"

    # 递进共振检测
    if daily_dir == Direction.Up:
        if m30_dir == Direction.Up or m5_dir == Direction.Up:
            return "多头递进共振"
    elif daily_dir == Direction.Down:
        if m30_dir == Direction.Down or m5_dir == Direction.Down:
            return "空头递进共振"

    # 无共振
    return "无共振"

# 使用
resonance = detect_resonance(czsc_daily, czsc_30m, czsc_5m)
print(f"共振状态: {resonance}")
```

## 五、多级别策略开发

### 5.1 策略模板

```python
from czsc.strategies import CzscStrategyBase
from czsc.core import Position, Event, Operate

class MultiLevelStrategy(CzscStrategyBase):
    """多级别共振策略"""

    @property
    def positions(self) -> List[Position]:
        # 大级别：日线趋势判断
        daily_trend = Event(
            name="日线趋势",
            operate=Operate.LO,
            signals_all=[
                "日线_cxt_bi_trend_向上_0",
            ],
            signals_any=[],
            signals_not=[]
        )

        # 中级别：30分钟买点
        m30_buy = Event(
            name="30分钟买点",
            operate=Operate.LO,
            signals_all=[],
            signals_any=[
                "30分钟_cxt_first_buy_符合_0",
                "30分钟_cxt_second_buy_符合_0",
            ],
            signals_not=[]
        )

        # 小级别：5分钟精准入场
        m5_entry = Event(
            name="5分钟入场",
            operate=Operate.LO,
            signals_all=[
                "5分钟_cxt_fx_power_强_0",
            ],
            signals_any=[],
            signals_not=[]
        )

        position = Position(
            name="多级别共振策略",
            symbol=self.symbol,
            opens=[daily_trend, m30_buy, m5_entry],
            stops=[],
            exits=[],
            holds=[],
            timeout=30,
        )

        return [position]
```

### 5.2 分级入场策略

```python
class StagedEntryStrategy(CzscStrategyBase):
    """分级入场策略

    第一批：大级别信号
    第二批：中级别确认
    第三批：小级别精准
    """

    @property
    def positions(self) -> List[Position]:
        # 第一批：50%仓位
        pos1 = Position(
            name="第一批建仓",
            symbol=self.symbol,
            opens=[Event(name="大级别信号", operate=Operate.LO, ...)],
            holds=[],
            timeout=30,
        )

        # 第二批：30%仓位
        pos2 = Position(
            name="第二批加仓",
            symbol=self.symbol,
            opens=[Event(name="中级别确认", operate=Operate.LO, ...)],
            holds=[],
            timeout=20,
        )

        # 第三批：20%仓位
        pos3 = Position(
            name="第三批加仓",
            symbol=self.symbol,
            opens=[Event(name="小级别精准", operate=Operate.LO, ...)],
            holds=[],
            timeout=10,
        )

        return [pos1, pos2, pos3]
```

### 5.3 级别切换策略

```python
class LevelSwitchStrategy(CzscStrategyBase):
    """级别切换策略

    根据市场状态切换分析级别
    """

    def get_active_level(self, czsc_daily, czsc_30m, czsc_5m):
        """获取当前活跃级别"""
        # 震荡市：关注小级别
        if self.is_consolidation(czsc_daily):
            return "5分钟"

        # 趋势市：关注大级别
        if self.is_trending(czsc_daily):
            return "日线"

        # 过渡市：关注中级别
        return "30分钟"

    def is_consolidation(self, czsc_obj):
        """判断是否震荡"""
        if len(czsc_obj.zs_list) > 0:
            zs = czsc_obj.zs_list[-1]
            # 在中枢内
            return True
        return False

    def is_trending(self, czsc_obj):
        """判断是否有趋势"""
        if len(czsc_obj.bi_list) >= 3:
            # 连续三笔同向
            last_three = czsc_obj.bi_list[-3:]
            return all(b.direction == last_three[0].direction for b in last_three)
        return False
```

## 六、常见问题

### Q1: 应该选择哪些级别？

A: 根据交易风格：

```python
"""
日内交易：
- 推荐：1分钟 + 5分钟 + 30分钟
- 特点：敏感度高，交易频繁

波段交易：
- 推荐：5分钟 + 30分钟 + 日线
- 特点：平衡灵敏度和稳定性

趋势交易：
- 推荐：30分钟 + 日线 + 周线
- 特点：稳定性高，交易次数少
"""
```

### Q2: 级别冲突怎么办？

A: 大级别优先：

```python
"""
级别冲突处理原则：

1. 大级别定方向
   - 大级别向上，只做多
   - 大级别向下，只做空

2. 小级别找位置
   - 在大级别方向内寻找小级别机会

3. 等待共振
   - 小级别与大级别共振时入场
"""
```

### Q3: 如何避免过拟合？

A: 保持简单：

```python
"""
多级别策略简化原则：

1. 限制级别数量
   - 不超过3个级别

2. 简化共振条件
   - 关注方向一致性
   - 避免复杂条件组合

3. 固定级别组合
   - 不要频繁调整级别
"""
```
