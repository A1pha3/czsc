## 一、信号函数概述

### 1.1 什么是信号函数？

**信号函数**是 CZSC 系统的基础单元，用于判断市场在某种特定状态下的表现。

```python
"""
信号函数的本质：一个将 CZSC 对象转换为信号字典的函数

输入：CZSC 对象 + 参数
输出：OrderedDict（信号字典）

示例：
输入：30分钟K线的CZSC对象，di=1
输出：{'30分钟_cxt_fx_power_强_0': '满足'}
"""
```

### 1.2 信号函数的分类

| 类别 | 模块 | 描述 | 示例函数 |
|------|------|------|----------|
| **技术指标信号** | `tas` | 基于传统技术指标 | `tas_ma_base`, `tas_macd_base` |
| **缠论形态信号** | `cxt` | 基于缠论形态 | `cxt_fx_power`, `cxt_third_buy` |
| **K线特征信号** | `bar` | 基于单根或多根K线 | `bar_end`, `bar_zdt` |
| **成交量信号** | `vol` | 基于成交量分析 | `vol_double_ma`, `vol_ti_suo` |
| **持仓信号** | `pos` | 基于持仓状态 | `pos_fx_stop`, `pos_ma` |
| **自定义信号** | `zdy` | 用户自定义逻辑 | `zdy_stop_loss`, `zdy_macd_bc` |

## 二、命名规范

### 2.1 标准命名格式

```python
"""
信号函数命名格式：{模块}_{特征}_{版本}

模块命名：
- tas: 技术指标（Technical Analysis Signals）
- cxt: 缠论形态（Chan Theory）
- bar: K线特征（Bar）
- vol: 成交量（Volume）
- pos: 持仓（Position）
- zdy: 自定义（Zi Ding Yi）

版本命名：
- V + YYMMDD 格式
- 例如：V221101 表示 2022年11月1日版本

示例：
- tas_ma_base_V221101      # 技术指标-均线-基础
- cxt_fx_power_V221107     # 缠论形态-分型力度
- bar_end_V221211          # K线特征-结束
"""
```

### 2.2 信号键命名格式

```python
"""
信号键（Signal Key）格式：{频率}_{特征}_{参数值}_{状态码}

频率：1分钟/5分钟/15分钟/30分钟/60分钟/日线/周线/月线
特征：信号的具体名称
参数值：信号的关键参数
状态码：0=默认，可扩展

示例：
- "30分钟_tas_ma_base_满足_0"
- "日线_cxt_fx_power_强_0"
- "5分钟_vol_double_ma_金叉_0"
"""
```

### 2.3 版本管理建议

```python
"""
版本更新原则：

1. 逻辑不变，性能优化 → 不升级版本
2. 参数新增，向后兼容 → 不升级版本
3. 逻辑变更，不兼容 → 升级版本
4. 废弃信号 → 标记 deprecated

版本迭代示例：
tas_ma_base_V221101  →  初始版本
tas_ma_base_V230313  →  逻辑优化，升级版本
tas_ma_base_V221101  →  保持兼容，两版本共存
"""
```

### 3.2 技术指标信号模板

```python
from collections import OrderedDict
from czsc.core import CZSC
from czsc.utils import update_ma_cache

def tas_your_indicator_V221101(c: CZSC, **kwargs) -> OrderedDict:
    """技术指标信号模板

    示例：基于均线的信号

    参数：
        c: CZSC 对象
        di: int, 偏移量
        ma_type: str, 均线类型（SMA/EMA）
        timeperiod: int, 均线周期
    """
    # 1. 获取参数
    di = kwargs.get('di', 1)
    ma_type = kwargs.get('ma_type', 'SMA')
    timeperiod = kwargs.get('timeperiod', 5)

    # 2. 数据校验
    if len(c.bars_raw) < di + timeperiod:
        return OrderedDict()

    # 3. 更新缓存（计算指标）
    update_ma_cache(c, ma_type=ma_type, timeperiod=timeperiod)

    # 4. 获取指标值
    cache_key = f"MA_{ma_type}_{timeperiod}"
    ma_values = [bar.cache.get(cache_key, 0) for bar in c.bars_raw]

    # 5. 信号判断
    current_ma = ma_values[-di]
    prev_ma = ma_values[-di - 1]

    if current_ma > prev_ma:
        v = "多头"
    else:
        v = "空头"

    # 6. 返回信号
    s = {
        f"{c.freq.value}_tas_your_indicator_{ma_type}_{timeperiod}_{v}_0": v
    }
    return s
```

from czsc.core import CZSC

def cxt_your_pattern_V221101(c: CZSC, **kwargs) -> OrderedDict:

### 3.4 K线特征信号模板

```python
from collections import OrderedDict
from czsc.core import CZSC

def bar_your_feature_V221101(c: CZSC, **kwargs) -> OrderedDict:
    """K线特征信号模板

    示例：基于K线实体的信号
    """
    # 1. 获取参数
    di = kwargs.get('di', 1)
    threshold = kwargs.get('threshold', 0.02)  # 2%

    # 2. 数据校验
    if len(c.bars_raw) < di:
        return OrderedDict()

    # 3. 获取K线
    bar = c.bars_raw[-di]

    # 4. 计算特征
    body_ratio = bar.solid / (bar.high - bar.low) if bar.high > bar.low else 0

    # 5. 信号判断
    if body_ratio > threshold:
        v = "大实体"
    else:
        v = "小实体"

    # 6. 返回信号
    s = {
        f"{c.freq.value}_bar_your_feature_{v}_0": v,
        f"{c.freq.value}_bar_body_ratio_{body_ratio:.4f}_0": body_ratio,
    }
    return s
```

### 3.5 成交量信号模板

```python
from collections import OrderedDict
from czsc.core import CZSC
from czsc.utils import update_ma_cache
import numpy as np

def vol_your_feature_V221101(c: CZSC, **kwargs) -> OrderedDict:
    """成交量信号模板

    示例：成交量放大的信号
    """
    # 1. 获取参数
    di = kwargs.get('di', 1)
    periods = kwargs.get('periods', 5)
    threshold = kwargs.get('threshold', 1.5)  # 1.5倍

    # 2. 数据校验
    if len(c.bars_raw) < di + periods:
        return OrderedDict()

    # 3. 获取成交量
    vols = [bar.vol for bar in c.bars_raw[-(di + periods):]
    current_vol = vols[-1]
    avg_vol = np.mean(vols[:-1])

    # 4. 信号判断
    if current_vol > avg_vol * threshold:
        v = "放量"
    elif current_vol < avg_vol / threshold:
        v = "缩量"
    else:
        v = "平量"

    # 5. 返回信号
    s = {
        f"{c.freq.value}_vol_your_feature_{v}_0": v,
        f"{c.freq.value}_vol_ratio_{current_vol/avg_vol:.2f}_0": current_vol / avg_vol,
    }
    return s
```

## 四、信号函数开发实践

### 4.1 实例一：MACD 底背驰信号

```python
from collections import OrderedDict
from czsc.core import CZSC
from czsc.utils import update_macd_cache

def tas_macd_bottom_divergence_V240228(c: CZSC, **kwargs) -> OrderedDict:
    """MACD底背驰信号

    判断条件：
    1. 价格创新低
    2. MACD柱子不创新低
    3. 至少有两个低点进行比较

    参数：
        c: CZSC 对象
        di: int, 偏移量，默认1
        n: int, 比较的笔数量，默认3
    """
    di = kwargs.get('di', 1)
    n = kwargs.get('n', 3)

    # 数据校验
    if len(c.bi_list) < n + di:
        return OrderedDict()

    # 更新MACD缓存
    update_macd_cache(c, fastperiod=12, slowperiod=26, signalperiod=9)

    # 获取最近的n个笔
    bis = c.bi_list[-(n + di):-di] if di > 0 else c.bi_list[-n:]

    # 找到低点笔（向下笔的终点是底分型）
    low_bis = [bi for bi in bis if bi.direction == Direction.DOWN]
    if len(low_bis) < 2:
        return OrderedDict()

    # 取最后两个低点笔
    bi_recent = low_bis[-1]
    bi_prev = low_bis[-2]

    # 获取对应的MACD值
    # 注意：需要找到笔结束时间对应的K线索引
    idx_recent = next((i for i, bar in enumerate(c.bars_raw)
                       if bar.dt == bi_recent.fx_b.dt), -1)
    idx_prev = next((i for i, bar in enumerate(c.bars_raw)
                     if bar.dt == bi_prev.fx_b.dt), -1)

    if idx_recent < 0 or idx_prev < 0:
        return OrderedDict()

    macd_recent = c.bars_raw[idx_recent].cache.get('MACD', 0)
    macd_prev = c.bars_raw[idx_prev].cache.get('MACD', 0)

    # 判断底背驰
    price_divergence = bi_recent.fx_b.fx < bi_prev.fx_b.fx  # 价格创新低
    macd_divergence = macd_recent >= macd_prev  # MACD不创新低

    if price_divergence and macd_divergence:
        v = "底背驰"
        s = {f"{c.freq.value}_tas_macd_bottom_divergence_{v}_0": v}
    else:
        s = {}

    return s
```

### 4.2 实例二：三买信号

```python
from collections import OrderedDict
from czsc.core import CZSC
from czsc.py.enum import Direction, Mark

def cxt_third_buy_V240228(c: CZSC, **kwargs) -> OrderedDict:
    """缠论三买信号

    判断条件：
    1. 有日线级别中枢
    2. 最新笔是向上笔
    3. 向上笔的起点的价格 > 中枢高点（ZG）

    参数：
        c: CZSC 对象
        di: int, 偏移量，默认1
    """
    di = kwargs.get('di', 1)

    # 数据校验
    if len(c.zs_list) < 1 or len(c.bi_list) < di:
        return OrderedDict()

    # 获取最新中枢
    zs = c.zs_list[-1]
    # 获取最新笔
    bi = c.bi_list[-di]

    # 三买条件
    condition1 = bi.direction == Direction.UP  # 向上笔
    condition2 = bi.fx_a.fx > zs.zg  # 起点价格突破中枢高点

    if condition1 and condition2:
        v = "符合"
        s = {f"{c.freq.value}_cxt_third_buy_{v}_0": v}
    else:
        s = {}

    return s
```

### 4.3 实例三：涨停板特征信号

```python
from collections import OrderedDict
from czsc.core import CZSC

def bar_limit_up_V240504(c: CZSC, **kwargs) -> OrderedDict:
    """涨停板信号

    判断条件：
    1. 涨幅 >= 9.9%（考虑浮点误差）
    2. 收盘价 = 最高价

    参数：
        c: CZSC 对象
        di: int, 偏移量，默认1
        threshold: float, 涨停阈值，默认0.099
    """
    di = kwargs.get('di', 1)
    threshold = kwargs.get('threshold', 0.099)

    # 数据校验
    if len(c.bars_raw) < di:
        return OrderedDict()

    bar = c.bars_raw[-di]

    # 计算涨幅
    if bar.open > 0:
        change_pct = (bar.close - bar.open) / bar.open
    else:
        return OrderedDict()

    # 涨停判断
    is_limit_up = (change_pct >= threshold) and (bar.close >= bar.high - 0.001)

    if is_limit_up:
        v = "涨停"
        s = {
            f"{c.freq.value}_bar_limit_up_{v}_0": v,
            f"{c.freq.value}_bar_change_{change_pct:.4f}_0": change_pct,
        }
    else:
        s = {}

    return s
```

### 4.4 实例四：多周期共振信号

```python
from collections import OrderedDict
from czsc.traders import CzscSignals

def multi_timeframe_resonance(signals_dict: dict, **kwargs) -> OrderedDict:
    """多周期共振信号

    判断条件：多个周期的信号方向一致

    参数：
        signals_dict: 各周期信号字典
            key: 周期名称（如 '30分钟', '日线'）
            value: 该周期的信号字典
        min_periods: int, 最少周期数，默认2
    """
    min_periods = kwargs.get('min_periods', 2)

    if len(signals_dict) < min_periods:
        return OrderedDict()

    # 统计做多/做空信号数量
    long_count = 0
    short_count = 0

    for period, signals in signals_dict.items():
        # 检查是否有做多信号
        if any('多头' in str(v) or '买入' in str(v) or '三买' in str(v)
               for v in signals.values()):
            long_count += 1
        # 检查是否有做空信号
        if any('空头' in str(v) or '卖出' in str(v) or '三卖' in str(v)
               for v in signals.values()):
            short_count += 1

    # 判断共振
    if long_count >= min_periods:
        v = "做多共振"
        s = {f"multi_timeframe_resonance_{v}_0": v}
    elif short_count >= min_periods:
        v = "做空共振"
        s = {f"multi_timeframe_resonance_{v}_0": v}
    else:
        s = {}

    return s
```
