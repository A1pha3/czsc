## 三、并行计算

### 3.1 多进程回测

```python
from concurrent.futures import ProcessPoolExecutor
import pandas as pd

def backtest_single_symbol(symbol):
    """单个品种回测"""
    # 准备数据
    bars = get_bars(symbol)

    # 执行回测
    trader = CzscTrader(strategy=MyStrategy(symbol=symbol))
    trader.replay(bars)

    # 返回结果
    return {
        'symbol': symbol,
        'pnl': sum(p.pnl for p in trader.positions[0].pairs),
        'count': len(trader.positions[0].pairs),
    }

# 多品种并行回测
symbols = ['000001', '000002', '600000', '600036', '000001']

with ProcessPoolExecutor(max_workers=4) as executor:
    results = list(executor.map(backtest_single_symbol, symbols))

# 分析结果
df_results = pd.DataFrame(results)
print(df_results)
```

## 四、算法优化

### 4.1 使用 Rust 版本

```python
# Rust 版本性能对比

# Python 版本
import os
os.environ['CZSC_USE_PYTHON'] = '1'
from czsc.core import CZSC as CZSC_PY

# Rust 版本（默认）
os.environ['CZSC_USE_PYTHON'] = '0'
from czsc.core import CZSC as CZSC_RS

# 性能测试
import time

bars = get_bars('000001', '1分钟', '20240101', '20240131')

# Python 版本
t0 = time.time()
czsc_py = CZSC_PY(bars)
t_py = time.time() - t0

# Rust 版本
t0 = time.time()
czsc_rs = CZSC_RS(bars)
t_rs = time.time() - t0

print(f"Python: {t_py:.3f}s, Rust: {t_rs:.3f}s, 加速: {t_py/t_rs:.1f}x")
```

### 4.2 批量处理

```python
# 不推荐：逐个处理
for bar in bars:
    process_bar(bar)  # 每次都调用函数

# 推荐：批量处理
def process_bars_batch(bars, batch_size=100):
    """批量处理K线"""
    for i in range(0, len(bars), batch_size):
        batch = bars[i:i+batch_size]
        # 批量处理
        results = [process_bar(bar) for bar in batch]
        yield results
```

### 5.3 使用生成器

```python

for bar in bars:
    result = process_bar(bar)
    results.append(result)  # 占用大量内存

# 推荐：生成器
def process_bars(bars):
    for bar in bars:
        yield process_bar(bar)

for result in process_bars(bars):
```

## 七、常见问题

### Q1: 缓存占用空间过大？

A: 定期清理：

```python
from czsc import clear_expired_cache
from czsc.utils import home_path

# 清理30天前的缓存（path 必选，max_age 单位为秒）
clear_expired_cache(home_path, max_age=3600 * 24 * 30)

# 也可以手动检查缓存大小后决定是否清理
from czsc.utils import get_dir_size
size_mb = get_dir_size(home_path) / 1024 / 1024
if size_mb > 500:
    clear_expired_cache(home_path, max_age=3600 * 24 * 7)  # 清理7天前的缓存
```

### Q2: 并行计算没有加速？

A: 检查以下因素：

```python
"""
并行计算问题：

1. GIL 限制
   - 使用多进程而非多线程
   - ProcessPoolExecutor

2. 进程启动开销
   - 数据量要足够大
   - 减少进程数

3. 内存争用
   - 避免共享大对象
   - 使用独立数据
"""
```

### Q3: 如何选择批量大小？

A: 根据实际情况调整：

```python
"""
批量大小选择：

1. 小批量（10-100）
   - 适合实时处理
   - 低延迟

2. 中批量（100-1000）
   - 适合一般回测
   - 平衡性能

3. 大批量（1000+）
   - 适合离线处理
   - 高吞吐量
"""
```
