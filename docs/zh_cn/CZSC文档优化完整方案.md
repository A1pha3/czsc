### 3. 术语词汇表

#### 核心术语定义表

| 术语 | 英文 | 定义 | 示例 |
|------|------|------|------|
| 分型 | Fractal | 由5根K线组成，中间K线为最高/最低的形态 | 顶分型(FX_G)、底分型(FX_D) |
| 笔 | Stroke | 两个不同方向分型之间的K线序列 | 向上笔(BI_UP)、向下笔(BI_DOWN) |
| 中枢 | Hub/Center | 两条笔形成的价格震荡区间 | ZS_zg(高点)、ZS_zd(低点) |
| 级别 | Level | 时间周期维度 | 日线、30分钟、5分钟等 |
| 包含关系 | Inclusion | 两根K线高低点被另一根包含 | 向上包含、向下包含 |
| NewBar | Processed Bar | 去除包含关系后的K线 | 原K线合并处理后的结果 |
| 信号 | Signal | 特定市场状态的判断函数 | cxt_third_buy、tas_macd_base |
| 事件 | Event | 多个信号通过AND/OR/NOT组合的交易条件 | 开仓事件、止损事件 |
| 持仓策略 | Position | 包含开/平/止损逻辑的完整交易规则 | 三买多头策略 |
| 回测 | Backtesting | 用历史数据验证策略性能 | CTAResearch、DummyBacktest |
| 过拟合 | Overfitting | 策略过度优化导致历史表现好但实盘差 | 参数过多、数据量少 |
| 共振 | Resonance | 多个级别同时出现同向信号 | 日线上升+30分钟买点 |
| 性能优化 | Performance | 减少计算时间或内存占用 | 缓存、Rust实现 |

#### 在每份文档中的应用
- 文档第一次出现新术语时，使用术语表中的定义
- 复杂术语添加"为什么"的解释
- 使用"→"表示关系链：分型→笔→中枢→交易点

  ```python
  def bar_limit_up_V240504(c: CZSC, **kwargs) -> OrderedDict:
      """涨停板信号（完整版）
      
      判断条件：
      1. 涨幅 >= 9.9%（考虑浮点误差）
      2. 收盘价 = 最高价
      
      参数模板："{freq}_D{di}涨停板V240504"
      
      返回值示例：
      - '15分钟_D1涨停板V240504_涨停_0'
      - '15分钟_D1涨停板V240504_跌停_0'
      """
      di = kwargs.get('di', 1)
      threshold = kwargs.get('threshold', 0.099)
      
      if len(c.bars_raw) < di:
          return OrderedDict()
      
      bar = c.bars_raw[-di]
      
      if bar.open > 0:
          change_pct = (bar.close - bar.open) / bar.open
      else:
          return OrderedDict()
      
      # 涨停判断
      is_limit_up = (change_pct >= threshold) and (bar.close >= bar.high - 0.001)
      is_limit_down = (change_pct <= -threshold) and (bar.close <= bar.low + 0.001)
      
      result = OrderedDict()
      if is_limit_up:
          result[f"{c.freq.value}_bar_limit_up_{di}涨停_0"] = "涨停"
      elif is_limit_down:
          result[f"{c.freq.value}_bar_limit_up_{di}跌停_0"] = "跌停"
      else:
          result[f"{c.freq.value}_bar_limit_up_{di}无_0"] = "无"
      
      return result
  ```

  ### 3.3 共振强度评分算法
  
  ```python
  def calculate_resonance_strength(signals_dict, direction='UP'):
      """
      计算多级别共振强度
      
      参数：
          signals_dict: 各级别信号字典
          direction: 共振方向 'UP' 或 'DOWN'
      
      返回：
          score: 0-10 的共振强度评分
      """
      
      # 级别权重（大级别更重要）
      level_weights = {
          '月线': 0.40,
          '周线': 0.30,
          '日线': 0.25,
          '60分钟': 0.20,
          '30分钟': 0.15,
          '15分钟': 0.10,
          '5分钟': 0.08,
          '1分钟': 0.05,
      }
      
      # 信号强度评分
      signal_strengths = {
          '强': 1.0,      # 分型力度强
          '中': 0.7,      # 分型力度中
          '弱': 0.4,      # 分型力度弱
          '符合': 1.0,    # 明确买卖信号
          '预兆': 0.6,    # 预兆信号
          '继续': 0.8,    # 趋势继续（同向时加分）
      }
      
      # 计算总体共振强度
      total_score = 0
      participating_levels = 0
      conflicting_signals = 0
      
      for level, signals in signals_dict.items():
          if not signals:
              continue
          
          level_weight = level_weights.get(level, 0.05)
          
          for signal_key, signal_value in signals.items():
              # 检查是否同向
              if direction in signal_key or direction.lower() in str(signal_value).lower():
                  signal_strength = signal_strengths.get(signal_value, 0.5)
                  total_score += level_weight * signal_strength * 10
                  participating_levels += 1
              else:
                  # 反向信号冲突
                  conflicting_signals += 1
      
      # 冲突减分
      if conflicting_signals > 0:
          total_score *= (1 - 0.3 * conflicting_signals)
      
      # 参与级别多时获得额外奖励
      if participating_levels >= 3:
          total_score *= 1.1
      elif participating_levels < 2:
          total_score *= 0.8
      
      return min(10, max(0, total_score))
  
  # 使用示例
  signals_dict = {
      '日线': {'日线_cxt_third_buy_符合_0': '符合', '日线_cxt_fx_power_强_0': '强'},
      '30分钟': {'30分钟_cxt_first_buy_符合_0': '符合'},
      '5分钟': {'5分钟_cxt_fx_power_强_0': '强'},
  }
  
  score = calculate_resonance_strength(signals_dict, direction='UP')
  print(f"共振强度: {score:.1f}/10")  # → 9.2/10 (三级共振)
  ```
