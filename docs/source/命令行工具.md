# 命令行工具

CZSC 提供了一些命令行工具，方便用户快速执行常用操作。

## 安装命令行工具

```bash
# 安装 CZSC 时会自动安装命令行工具
pip install czsc
```

## 可用命令

### czsc-init

初始化一个新的 CZSC 策略项目。

```bash
czsc-init my_strategy
```

这将创建一个包含以下结构的策略项目：

```
my_strategy/
├── config/           # 配置文件目录
├── signals/          # 自定义信号
├── strategies/       # 策略定义
├── data/             # 数据目录
└── main.py           # 主程序入口
```

### czsc-backtest

执行策略回测。

```bash
czsc-backtest --config config/backtest.yaml --symbol 000001
```

参数说明：
- `--config`: 配置文件路径
- `--symbol`: 交易品种代码
- `--start`: 回测开始日期 (默认: 20200101)
- `--end`: 回测结束日期 (默认: 当天)
- `--output`: 输出目录 (默认: ./outputs)

### czsc-analyze

分析 K 线数据并输出缠论分析结果。

```bash
czsc-analyze --data data/000001.csv --freq 30分钟
```

参数说明：
- `--data`: K 线数据文件路径（CSV 格式）
- `--freq`: K 线频率（1分钟、5分钟、30分钟、日线等）
- `--output`: 输出文件路径（可选）

### czsc-plot

生成 K 线图表和分析图表。

```bash
czsc-plot --data data/000001.csv --output kline_chart.html
```

参数说明：
- `--data`: K 线数据文件路径
- `--output`: 输出 HTML 文件路径
- `--type`: 图表类型（kline、backtest、heatmap）
- `--template`: 图表模板（plotly、plotly_dark）

## 配置文件

### 回测配置示例

```yaml
# backtest.yaml
symbol: "000001"
freq: "30分钟"
start_date: "2023-01-01"
end_date: "2023-12-31"

signals:
  - name: "czsc.signals.tas_ma_base_V221101"
    params:
      freq: "日线"
      di: 1
      ma_type: "SMA"
      timeperiod: 5

positions:
  - name: "笔非多即空"
    operate: "开多"
    signals_all:
      - "30分钟_D1_表里关系V230101_向上_任意_任意_0"
    timeout: 100
    stop_loss: 500

backtest:
  fee_rate: 0.0003
  initial_capital: 100000
```

## 使用示例

### 完整回测流程

```bash
# 1. 初始化项目
czsc-init my_strategy
cd my_strategy

# 2. 编辑配置文件
vim config/backtest.yaml

# 3. 执行回测
czsc-backtest --config config/backtest.yaml

# 4. 生成图表
czsc-plot --data outputs/000001.csv --type backtest

# 5. 查看结果
open outputs/backtest_report.html
```

### 批量回测

```bash
# 批量回测多个品种
for symbol in 000001 000002 000003; do
    czsc-backtest --symbol $symbol --output outputs/$symbol
done
```

## 常见问题

### Q: 命令找不到？

A: 确保 CZSC 已正确安装，并检查 Python Scripts 目录是否在 PATH 中。

```bash
# 检查安装
pip show czsc

# 重新安装
pip install czsc --upgrade --force-reinstall
```

### Q: 如何查看命令帮助？

A: 使用 `--help` 参数查看详细帮助信息。

```bash
czsc-backtest --help
czsc-analyze --help
czsc-plot --help
```

A: 将自定义信号文件放在 `signals/` 目录下，然后在配置文件中引用。

```python
# signals/my_signal.py
from collections import OrderedDict
from czsc.core import CZSC

def my_signal_V240101(c: CZSC, **kwargs) -> OrderedDict:
    # 信号实现
    return s
```

```yaml
# config/backtest.yaml
signals:
  - name: "signals.my_signal.my_signal_V240101"
    params:
      di: 1
```
