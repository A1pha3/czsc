# 开发日志

## 0.10.8 (2024)

### 新增功能
- 新增 `czsc/utils/analysis/` 分析工具模块
  - `stats.py`: 统计分析（日收益表现、回撤分析、PSI等）
  - `corr.py`: 相关性分析（NMI矩阵、截面IC等）
  - `events.py`: 事件分析（重叠检测等）

### 重构优化
- `czsc/utils/plotting/` 重构为模块化结构
  - `backtest.py`: 回测图表
  - `weight.py`: 权重分析图表
  - `kline.py`: K线图表
  - `common.py`: 通用常量和辅助函数
- 提供统一导入接口，保持向后兼容

### 文档改进
- 新增 `快速开始指南.md`
- 更新 `源码阅读指南.md`，添加新模块说明
- 更新 `项目核心工具.md`，更新API参考
- 完善 `README.md` 文档导航

## 0.9.7

* 新增 CzscSignals 用于生成信号
* 完成 Position + CzscTrader 的开发；此后的策略开发推荐使用 Position + CzscTrader 的方式，这样可以更好的控制仓位和风险
